# Azure Automation Tools

This image is based on alpine 3.8 and contains powershell, powershell azure module and azure cli

## Installation

```powershell
docker pull bczudek\azure-automation-tools:1.1
```

## Libraries

```text
alpine 3.8
powershell 6.2.2
powershell module: az 2.5.0
azure cli 2.0.70

```
